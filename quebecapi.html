Quebec API

The Quebec API provides real-time updates on power outages across Quebec:

  • Data updated every 5 minutes.
  • Search for outages by latitude, longitude, and distance.

Code Example

# Import necessary libraries
import requests

# API configuration
API_KEY = "your-api-key"
URL = "http://example.com/api"

# Example request
response = requests.get(URL, headers={"X-API-KEY": API_KEY})
print(response.json())
