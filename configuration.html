Configuration

To configure the Canadian Power Outages API, follow the steps below:

  • Step 1: Obtain your API key by emailing andrew@canadianpoweroutages.ca.
  • Step 2: Use the API key to authenticate your requests.
  • Step 3: Pass configuration parameters as query parameters in the request URL.
